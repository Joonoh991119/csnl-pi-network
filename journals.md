## Search Priority Order

When searching for papers for a researcher:
1. Search Tier 1-2 journals first using web search with site filters
2. If insufficient results, expand to Tier 3
3. Check Tier 4 conference proceedings for modeling/computational papers
4. Fall back to Tier 5 preprints only if nothing suitable found above
5. Always verify: is the paper actually published (not just submitted)?
